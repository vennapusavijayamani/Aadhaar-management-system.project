
# Aadhaar Management System

## Description
A simple Python-based Aadhaar Management System that performs CRUD operations using a dictionary.

## Features
- Create Record
- Read Record
- Update Record
- Delete Record
- Display All Records

## Concepts Used
- Python Dictionary
- Functions
- Conditional Statements
- Loops
- User Input
- CRUD Operations
